# 👻 GhostScreen-ahk

**Proteção Ativa e Transparente para Automações (RPA).**

Ferramenta em AutoHotkey que cria uma camada "click-through" sobre a tela, bloqueando interações humanas acidentais enquanto mantém a visibilidade total da execução do robô. Possui ajuste universal para múltiplos monitores, proteção por senha e atalho de emergência.

---

## ✨ Funcionalidades
* **Bloqueio Total/Parcial:** Escolha bloquear todos os monitores ou apenas um específico.
* **Interface Invisível:** O usuário vê a automação, mas não consegue interferir no mouse/teclado.
* **Segurança Personalizada:** Altere sua senha diretamente pela interface.
* **Ajuste Universal:** Funciona em qualquer resolução (Notebook, Full HD, 4K).

## 🚀 Como usar
1. Baixe o arquivo `GhostScreen.exe` (ou execute o `.ahk`).
2. A senha padrão inicial é `12345`.
3. Escolha o monitor e clique em **Proteger Tela**.
4. Para sair, pressione **CTRL + Q** e digite sua senha.

## 🛠️ Requisitos
* Windows 10 ou 11.
* [AutoHotkey](https://www.autohotkey.com/) (apenas se for rodar o código fonte).
